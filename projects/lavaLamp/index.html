CPSC 8170: Final Project Proposal

Overview

For my final project, I hope to simulate the behavior of a lava lamp in OpenGL. To simplify the animation, I will model the fluid using spherical particles or blobs known as metaballs, and then approximate their movement using the marching cubes algorithm.

Method

The proposed project is essentially an extension of the particles assignment. I will first try to implement blobby objects in 2D using the marching squares algorithm, and then try to do the same in 3D using marching cubes. The marching squares algorithm generates an approximation for a contour line of a 2D scalar grid by examining the corners of each cell. In order to display the contour line, every case can be enumerated as in Figure 3. Therefore in my implementation, I will have to first specify a region to subdivide into individual cells (or voxels in 3D). Next using the marching squares algorithms, I will draw the contour lines for circular objects moving within the region. After doing this at a large enough grid resolution, the result should be a blobby effect (Figure 1). Finally, I will use this same process in 3D, improve the animation by constraining the objects within a lava lamp using collision detection, and add lighting.
